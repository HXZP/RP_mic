# STM32_TIMER_ADC

MCU: STM32F103C8Tx

## DevelopTools

* STM32CubeMX
* arm-none-eabi-gcc
* CLion(CMake)
* OpenOCD
* ST-LINK

## Install Requirements(for macOS)

* config OpenOCD and ST-LINK
```bash
brew install openocd
brew install stlink
brew cask install gcc-arm-embedded
```
